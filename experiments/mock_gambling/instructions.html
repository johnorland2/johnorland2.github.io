How to use the controls:

1. There's an orange rectangle that has a "play" label. It is intended as a clickable button; a click triggers a transition in the application from the menu screen to the game-play screen. You can click on the "play" button.

2. In the game-play screen, there are round, orange elements that are intended as clickable buttons; clicks cause a transition in the application. Two of these elements are labeled "rules" and "spin". You can click "rules" to transition the interface to the "rules" description screen. You can click "spin" to cause the interface to consume your bet, spin the columns and possibly yield winnings. Two smaller, orange elements are labeled "-" and "+". These can be clicked in order to decrease and increase the bet respectively.

3. If you go to the "rules" description screen, you'll see another orange element in there. It has the "back" label. If you click it, the application transitions back to the main game-play screen.
