Instructions:

Overview:

Primarily, the application presents a sequence of interactive multiple choice questions. This interface could be used as part of a "trivia" game at a party. Alternatively, the interface could be adapted for use as a study/memory aid.

The user may use mouse clicks to interact with the page. Clicks are used to transition between application phases, and to submit answers to questions.

About how to click on stuff properly:

Look out for orange rectangles that show a pointer (hand) icon when the mouse is moved into hovering viscinity. These rectangles may be construed as buttons; a click triggers a transition in the application. The rectangle's textual label is a cue that suggests the behavior that's connected with a click. You might see other textual indications in the viscinity of these rectangles, and these indications further elucidate what's going on.

Description of phases:

1. The first type of phase is the "menu". In this phase, the user can see a "title" and a button that says, "play". Clicking the button transitions the application to a question phase.

2. The second type of phase is the "question" phase. The user can see a textual indication of the question's prompt and the question's category. Below these indications, there are multiple clickable buttons. Each button represents a possible answer to the question; the button's textual label conveys a possible answer to the question. You get one chance to answer a question. If you click on one answer, then your opportunity is spent.

After clicking an answer, the application is transitioned to a subphase that shows a textual indication that the answer was either correct or incorrect. This subphase also gives a button; the button either leads to another question phase, or leads to the summary phase if no further questions are available. Look for a textual cue that elucidates what's going on at this point.

3. The third (final) type of phase is the summary. In this phase, you can see a textual indication that explains how many questions were answered correctly.
